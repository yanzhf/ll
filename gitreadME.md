#git命令总结

###基础

####git的提交
> git clone 下载git文件  
> git add . 本地添加文件  
> git commit -m '' 提交文件到本地仓库  
> git push -u origin xxx 提交到远程分支  
> git tag xxx 打标签  
> git push --tags 提交标签  


#### git分支  
> git branch 查看当前的分支  
> git branch xxxx 创建分支  
> git checkout xxx 切换分支  
> git merge xxx 合并分支  
> git branch  -d xxx 删除本地分支  
> git push origin   --delete xxx 提交删除远程分支
